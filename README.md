# Estudo angular

Este projeto cria uma aplicação para cadastro de produtos, aplicando as operações de criação, leitura, atualização e exclusão de itens. 

## Aplicação CRUD

Este projeto utiliza:

- Angular na versão 17
- [Angular CLI][#1]
